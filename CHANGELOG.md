# Changelog

All notable changes to this project are documented here.

## [0.1.0] - 2026-06

Initial release.

### MCP tools (32)

- **Discovery:** `list_apps`, `get_app_info`, `list_versions`, `get_version_localizations`
- **App metadata:** `update_app_info_localization`, `create_app_info_localization`,
  `update_version_localization`, `create_version_localization`,
  `bulk_update_version_localizations`
- **App availability & price:** `set_app_availability`, `set_app_price`
- **Versions:** `create_version`, `update_version`
- **IAP:** `list_iaps`, `create_iap`, `update_iap_localization`,
  `set_iap_availability`, `set_iap_price`
- **Subscriptions:** `create_subscription_group`,
  `set_subscription_group_localization`, `create_subscription`,
  `set_subscription_availability`, `set_subscription_price`,
  `update_subscription_localization`
- **Screenshots:** `upload_screenshot`, `list_screenshot_sets`,
  `delete_screenshot`, `upload_product_review_screenshot`
- **Submission:** `create_review_submission`, `add_version_to_submission`,
  `submit_for_review`, `get_submission_status`

### Highlights

- Typed App Store Connect client generated by `oapi-codegen` from Apple's
  official OpenAPI spec (pruned to the used operations).
- ES256 JWT provider with caching/auto-refresh; secrets never logged.
- Domain validation (field limits, editable states) before any request.
- Idempotency + audit log via optional Postgres (no-op when disabled).
- Dry-run mode; normalized ASC error reporting.
- Config auto-loaded from `~/.config/asc-mcp/config.env` (and others);
  `-setup` / `-version` flags; actionable first-run guidance.
- Canonical end-to-end playbook in `docs/APP-PRODUCT-SETUP-FLOW.md`.
